# frontend

## How this was created

```
nvm install --lts
npx @vue/cli create frontend --default
npx @vue/cli add router
npx @vue/cli add apollo
```

## Project setup
```
npm install bootstrap@4.6.0 bootstrap-vue@2.21.2

```
